# online-learning-dropout-risk-analysis-powerbi
This project analyzes student engagement and dropout risk in online learning platforms using a Power BI dashboard. It visualizes key metrics such as login frequency, session duration, course progress, and quiz performance to identify patterns that help educators monitor engagement and improve student retention.

🎯 Problem Statement:
Online learning platforms generate large amounts of student interaction data. However, extracting meaningful insights from this data can be challenging.
Educational institutions often struggle to monitor student engagement effectively and identify learners who may be at risk of dropping out. Without proper data analysis and visualization tools, it becomes difficult to detect patterns in learning behavior.
This project addresses this challenge by building a data-driven analytics dashboard that visualizes student engagement metrics and highlights factors associated with dropout risk.

🛠 Tools & Technologies Used
Power BI Desktop – Data visualization and dashboard development
Dataset (CSV) – https://www.kaggle.com/datasets/jayjoshi37/online-learning-engagement-and-dropout-risk
Power Query – Data cleaning and transformation
Microsoft Excel – Data preprocessing 

📊 Dashboard Features
The Power BI dashboard provides multiple analytical views including:
Student engagement metrics
Dropout risk distribution
Login frequency analysis
Course progress vs session time analysis
Device usage insights
Course category engagement comparison
Forum interaction analysis

📈 Key Insights: 
Students with higher login frequency show lower dropout risk.
Increased session time correlates with better course progress.
Certain course categories show higher engagement levels.
Forum participation contributes to improved engagement.
Device usage patterns show most learners access courses through desktop and mobile devices.
